# tools

## 目的
提供可复用的工具能力与 RAG 检索。

## 模块概述
- **职责:** RAG 检索、信件生成等工具函数
- **状态:** 开发中
- **最后更新:** 2026-01-19

## 规范
### 需求: RAG 引用标注
**模块:** tools
返回带脚注标记与参考文献的回答。

#### 场景: 回答带引用
用户提问后，输出回答并标注引用来源。
- 引用需包含页面标题与 URL
- 引用粒度按检索块/段落去重
- 必须使用 LLM 生成答案并添加脚注
- 脚注使用数字编号并与资料编号对应
- 参考文献按标题与 URL 去重
- 支持从动态字段读取标题与 URL
